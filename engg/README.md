# Engineering Documentation

This directory holds internal product and engineering documents for s3q.

## Structure

- `product-requirements.md`: primary PRD for the project
- `design/`: living design documents
- `adr/`: architecture decision records
- `investigations/`: research notes and references
- `processes/`: engineering process notes
- `reviews/`: substantial review notes and test plans

## Writing Rule

Use `docs/` for external documentation and `engg/` for planning, decisions, and implementation notes.
